## INVENTORY HUB

### Overview

An application that helps businesses manage sales and staff as well as the products and services that they offer.

### Tech Stacks

The frontend application is built using the following tech stacks:

- React
- Zustand
- TypeScript
- Tailwind
- Horizon UI Template

### Views

- Registration
- Login
- Reset Password
- Dashboard
- User
- Stores
- Profiles
- Products
- Sales
- Profile
